Descripción del proyecto

Consultorio MEMITO es el sistema web creado para el consultorio médico "MEMITO", este sistema web es capaz de almacenar todos los datos relacionados a los pacientes, los medicamentos, los expedientes clínicos y las consultas. Además, posee un Dashboard en tiempo real que permite al médico supervisar y tomar decisiones relacionadas al consulatorio.

El proyecto a realizar fue la creación del sistema y la creación de la base de datos SQL para almacenar de manera correcta la información, todo esto basado en los requerimientos del dueño del consultorio.

Las funcionalidades que posee el sistema es la creación, actualización y eliminación de los datos de la base de datos, además, cada sección muestra en una tabla la información relevante y útil para el médico.

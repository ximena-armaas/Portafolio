Descripción del proyecto

Marketing Turistico Digital es un sistema web diseñado para los estudiantes del CONALEP, en especifico aquellos que están cursando la materia de Marketing Turistico.

El proyecto a realizar fue la creación del sistema a partir de las necesidades de los alumnos, el sistema tiene como objetivo ser una herramienta de estudio para los alumnos, por lo que, el sistema tiene toda la información del temario, además, implementa recursos didácticos para que los alumnos puedan aprender mientras se divierten.

Entre los recursos didácticos que se implementaron fueron juegos como "El ahorcado" y "La sopa de letras", Videos y Quizes. Cuando el alumno no acierte en alguna pregunta del quiz, se le recomiendan videos que pueden ayudar a retroalimentar los temas.

Por último, se creó una base de datos SQL para registrar a los alumnos y almacenar el porcentaje de avance que cada uno de ellos tuviera

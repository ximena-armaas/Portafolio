Descripción del proyecto

El PERD es el sistema de emisión de reconocimientos digitales de la Universidad Politécnica de Tlaxcala, este sistema tiene la capacidad de registrar eventos, usuarios, asistentes a eventos, diseños de los reconocimientos, prefijos de los reconocimientos entre otras cosas, para automatizar la tarea de emisión de reconocimientos digitales a los asistentes de estos.

El proyecto a realizar fue la actualización de este, entre las tareas que se hicieron fue la reestructuración de la base de datos, crear nuevas secciones al sistema, renovar la imagen del sistema y agregar nuevas funcionalidades al sistema. Las actualizaciones se hicieron con el framework "CodeIgniter" Entre las funcionalidades nuevas implementadas está el envio de correo electrónico automatico a la hora de registrar un usuario.
